From zero to app in minutes

Mulberry takes the promise of PhoneGap — mobile apps built with web technologies — and smooths over many of the “gotchas” that front-end developers confront as they venture into client-side application development in a mobile environment. When you use Mulberry to help you build mobile apps with web technologies, here are just a few of the things you get out of the box:

  • Ready-to-deploy apps to Android, iPhone, and iPad
  • Rapid in-browser iteration, eliminating the need for constant, time-consuming device builds
  • Over-the-air data updates that persist for offline use, keeping your app’s content current without requiring resubmission
  • Integration of common plugins and components
  • A complete framework for creating dynamic JavaScript applications

Mulberry provides all of this and more, giving you the building blocks you need to create a content-rich application that will work on iOS and Android, whether your user is online or off. With Mulberry’s clear patterns and simple APIs, you can develop a robust “single-page” application that you can immediately deploy to the iOS and Android app stores. When you’re ready to build your next app, your components, templates, and themes will plug easily into any application that uses the Mulberry framework.

A pluggable framework for building mobile apps

Mulberry is a powerful, pluggable framework for creating mobile applications using familiar HTML5 technologies. Use Mulberry’s built-in components to create, customize, and compile content-rich applications, or use Mulberry’s simple APIs to build components, interactions, layouts, and themes that you can share across your applications.

Powerful out of the box

Mulberry comes with built-in components, page layouts, and a simple theme that will get you up and running in a hurry, which means you can create your first Mulberry app in the time it takes to create and organize your content. When you’re ready to branch out, you can create your own components using Haml and Mulberry’s Javascript APIs. You can also take advantage of SASS to customize the default theme or create your own style from scratch.

An emphasis on smart reuse

The Mulberry framework was designed from the start with reuse in mind, with a focus on loose coupling and separation of concerns. Mulberry provides a solid base for custom development, with an emphasis on smart reuse within and across applications. It gives you the infrastructure you need — a router, models, views, offline storage, over-the-air data updates, and lots more — plus simple APIs for extending the framework with your own functionality. With Mulberry, creating and customizing apps becomes a reliably repeatable process.
